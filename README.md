___Project MC BUY___

Задание: 
Сверстать макет (адаптивно) с использованием стандартов HTML5 и CSS3. 

1). Верстка блоками (div) без использования таблиц. Таблицы в верстке используются только там,
где они необходимы - контент.

2). Кроссбраузерность
Firefox (6.0 и выше)
MSIE (11 и выше)
Opera (11 и выше)
Google Chrome (9 и выше)
Safari (6 и выше)
Mobile Safari

3). Верстать под HTML5,
Проверить работу валидатором
Валидатор <!DOCTYPE html>

4). Все стили выносить в css файлы

5). В css файлах не должно быть неиспользуемых стилей

6). CSS (включая фиксы и хаки) складировать в папке /css/
Картинки в папке /images/
Яваскрипты в папке /js/

7). Обязательно нужно прописать стили для:
+абзаца текста;
+иерархии заголовков трех уровней;
+ссылок + наведение;
+ненумерованный список;
+вложенный список;
+нумерованный список;
+иллюстрация в тексте статичной страницы;
+таблица;
+простая форма (все возможные input, а также button, textarea, select)
